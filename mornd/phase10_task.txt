## 设计

### 新增表

#### L3: semantic_knowledge

```sql
CREATE TABLE IF NOT EXISTS semantic_knowledge (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    subject     TEXT NOT NULL,          -- 主体
    relation    TEXT NOT NULL,          -- 关系（喜欢/不喜欢/擅长/不擅长/是/有/想要）
    object      TEXT NOT NULL,          -- 客体
    confidence  REAL NOT NULL DEFAULT 0.5,  -- 置信度 0.0-1.0
    source      TEXT NOT NULL DEFAULT 'chat',  -- 来源
    created_at  TEXT NOT NULL DEFAULT (datetime('now')),
    verified_at TEXT,                   -- 最后一次验证时间
    forgotten   INTEGER NOT NULL DEFAULT 0
)
```

索引：
- subject + relation
- object
- subject + relation + object UNIQUE

#### L4: personality_memory（只追加）

```sql
CREATE TABLE IF NOT EXISTS personality_memory (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    category    TEXT NOT NULL,          -- identity / belief / affirmation / narrative
    content     TEXT NOT NULL,          -- 内容
    importance  REAL NOT NULL DEFAULT 0.5,
    source      TEXT NOT NULL DEFAULT 'self_reflection',
    created_at  TEXT NOT NULL DEFAULT (datetime('now')),
    -- 没有 deleted/forgotten 字段——只追加不可删除
    -- 如需"推翻"一条记录，追加一条新的 statements 来覆盖
)
```

### 后见之明自动触发

在 `SelfReflection.deep_reflection` 中增加：每 10 次深度自省（约 50 分钟），
扫描所有超过 30 天未有后见之明的记忆，自动追加情感标签。

```python
async def _auto_hindsight(self):
    """自动对 30 天前的记忆追加后见之明标签"""
    cutoff = (datetime.now(timezone.utc) - timedelta(days=30)).isoformat()
    cursor = await self.db.execute("""
        SELECT * FROM capsules WHERE timestamp < ? AND source NOT IN ('self_reflection', 'security')
        ORDER BY emotion_score DESC LIMIT 3
    """, (cutoff,))
    rows = await cursor.fetchall()
    for row in rows:
        cap = self._decrypt_capsule(dict(row))
        current_tags = json.loads(cap.get("emotion_tag", "[]")) if cap.get("emotion_tag", "").startswith("[") else [cap.get("emotion_tag", "")]
        if "后见之明" not in current_tags:
            await self.add_hindsight_tag(cap["event_id"], "后见之明", cap["emotion_score"])
```
